# p2-cms-integration-server
## **List Endpoints**

List of Available Endpoints:
### **- Users**
- `POST /users/register`
- `POST /users/login`
- `POST /users/googleSignIn`

### **- Ip Location** 
- `GET /ip`

### **- Search Location** 
- `POST /search`

### **- Popular Location** 
- `GET /popular`
- `GET /popular/:popularId`

### **- Popular Location** 
- `POST /saved`
- `GET /saved`
- `GET /saved/:savedId`
- `DELETE /saved/:savedId`
